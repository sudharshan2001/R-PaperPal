# RP-Recommender

## Setup

#### Conda 
##### Clone this repository

```bash
  git clone https://github.com/sudharshan2001/Zebfi.git
  cd Zebfi
```

##### Create Environment using conda

```bash
  conda create -n zfish python=3.9.7
  activate zfish
```

##### Install the requirements
```bash
  pip install -r requirements.txt
```
